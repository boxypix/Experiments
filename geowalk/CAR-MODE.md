# Режим машины — настройки

**Единственное место для правок: `settings.json`**

После изменений: `npm run snapshot` (для `file://`) или перезагрузка при `npm start`.

**Клавиши:** `C` — переключить вид камеры (сзади ↔ сверху).

Спрайт `car_0.png`: **2400×400**, четыре кадра по **600 px** (0–2: сзади при повороте, 3: вид сверху).

---

## Камера (вид сзади)

| Параметр | По умолч. | Эффект |
|----------|-----------|--------|
| `carCameraHeight` | `7` | Высота камеры, м |
| `carCameraPitch` | `74` | Наклон вниз, ° |
| `carLookAheadM` | `10` | Точка взгляда впереди от машины, м |

## Камера (вид сверху, клавиша C)

| Параметр | По умолч. | Эффект |
|----------|-----------|--------|
| `carTopCameraHeight` | `14` | Выше и дальше |
| `carTopCameraPitch` | `15` | Наклон MapLibre: `0` = строго сверху, `90` = горизонт |
| `carTopLookAheadM` | `0` | Центр камеры на машине |
| `carTopZoom` | `18.5` | Масштаб карты (дальность вида) |

## Спрайт

| Параметр | По умолч. | Эффект |
|----------|-----------|--------|
| `carSpriteSrc` | `"images/car_0.png"` | Файл спрайт-листа |
| `carSpriteWidthPx` | `180` | Ширина на экране, px |
| `carSpriteTurnThresholdDeg` | `[15, 30]` | Пороги кадров 0–2, °/с |
| `carSpriteTopFrameIndex` | `3` | Кадр вида сверху (4-й) |

## Управление

| Параметр | По умолч. |
|----------|-----------|
| `carSpeed` | `55` |
| `carTurnSpeedDeg` | `110` |
| `carAccelSec` | `0.65` |
| `carBrakeSec` | `1.05` |
| `carHandbrakeSec` | `0.22` |
| `carTurnInertiaSec` | `0.5` |
